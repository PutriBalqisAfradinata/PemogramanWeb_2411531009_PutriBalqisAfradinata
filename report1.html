Report 1 — Instalasi Laravel

Pemrograman Web Semester 3 Putri Balqis Afradinata

I. Tujuan Praktikum

Setelah menyelesaikan praktikum ini, mahasiswa diharapkan mampu:

  • Memahami konsep dasar Laravel
  • Melakukan instalasi Laravel
  • Menjalankan project Laravel
  • Memahami struktur folder Laravel


II. Dasar Teori

Laravel merupakan framework PHP berbasis MVC yang digunakan untuk membangun aplikasi web modern dengan sintaks yang elegan dan terstruktur.



III. Langkah Kerja

  1. Menginstall Composer
  2. Membuka terminal / CMD
  3. Menjalankan perintah instalasi Laravel
  4. Membuat project baru Laravel
  5. Menjalankan local server
