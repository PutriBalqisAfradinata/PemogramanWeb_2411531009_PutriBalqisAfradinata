Website ini merupakan proyek akhir mata kuliah Desain Web, yang dibuat menggunakan HTML, CSS, dan Bootstrap 5. 
Website ini menampilkan informasi pribadi, pengalaman, serta kumpulan proyek yang pernah dibuat dalam bentuk portfolio yang responsif dan user-friendly.

Struktur Halaman:

    1.  Home
        Berisi hero section dengan perkenalan singkat dan tombol call-to-action.

    2.  About
        Menampilkan informasi pribadi, visi, misi, serta timeline perjalanan akademik dan skill menggunakan komponen accordion.

    3.  Portfolio
        Berisi grid portfolio dengan minimal 6 project, dilengkapi kategori, badge, filter tabs, serta modal detail dan carousel gambar.

    4.  Contact
        Menyediakan form kontak dengan validasi (client-side validation), alert message (success & error), kontak lengkap, dan Google Maps embed.

Fitur Utama:

    1.  Navbar responsive dan konsisten di setiap halaman

    2.  Carousel/slider pada detail portfolio

    3.  Grid system & layout Bootstrap yang responsif

    4.  Custom CSS untuk tema pink modern & clean

    5.  Form validation dengan alert success & error

    6.  Modal detail project + tombol GitHub

    7.  Footer dengan informasi, quick links, dan social media

Tujuan Website:

Website ini dibuat sebagai media personal branding dan wadah untuk menampilkan kemampuan dalam web design serta pengembangan antarmuka menggunakan Bootstrap.
